# OpenGL 实时光线追踪渲染器 
主要内容：使用 OpenGL 实现了一个实时全局光照的光线追踪渲染器。<br>
功能：在片段着色器中实现了路径追踪算法，支持漫反射与镜面材质，使用联合双边滤波以及帧间重投影进行降噪，康奈尔盒场景达到 30+FPS 以及较好的渲染质量。 <br>
